# WPAdmin

**WPAdmin** — настольное приложение для Windows (WinUI 3) для учёта сайтов, хостингов и доменов, работы с **REG.API**, мониторинга доступности и напоминаний. В этом репозитории публикуются только **готовые сборки**; исходный код не выкладывается.

**Скачать:** раздел [**Releases**](https://github.com/ljosefl/WPAdmin-Releases/releases) · [**Последняя версия (latest)**](https://github.com/ljosefl/WPAdmin-Releases/releases/latest)

## Что умеет программа

- Учёт **проектов (сайтов)** и привязка к хостингу и аккаунту провайдера  
- Учёт **доменов**, синхронизация сроков и данных через REG.API  
- **Мониторинг доступности** (HTTP), уведомления, журнал проверок  
- **RSS REG.RU:** новости провайдера после синхронизации с аккаунтов  
- Локальная **база данных** (Microsoft SQL Server), настройки интеграций, работа из области уведомлений Windows  

## Требования

- **ОС:** Windows 10 (1809+) или Windows 11, **x64**  
- **БД:** Microsoft SQL Server — строка подключения задаётся в настройках приложения  

## Установка

1. Скачайте архив **`WPAdmin-*-win-x64.zip`** из выбранного [релиза](https://github.com/ljosefl/WPAdmin-Releases/releases).  
2. Распакуйте в удобную папку.  
3. Запустите **`WPAdmin.App.exe`**.  

В архиве есть каталог **`SQL/`** — скрипты создания схемы базы данных для SQL Server.

## Обновления

В приложении: **Настройки → Интеграции** — укажите репозиторий **`ljosefl/WPAdmin-Releases`**. Проверка обновлений идёт через GitHub Releases (`/latest`); версия сравнивается с тегом релиза (например **`v1.0.1`**).
