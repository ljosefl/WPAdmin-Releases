# WPAdmin — выпуски

Готовые сборки для **Windows x64** публикуются в разделе [**Releases**](https://github.com/ljosefl/WPAdmin-Releases/releases).

**Актуальная версия:** [Latest](https://github.com/ljosefl/WPAdmin-Releases/releases/latest).

### Установка

1. Скачайте архив WPAdmin-*-win-x64.zip из релиза.
2. Распакуйте в удобную папку.
3. Запустите WPAdmin.App.exe.

В архиве есть каталог **SQL/** — скрипты создания схемы БД для SQL Server.

Исходный код здесь не хранится (приватная разработка).

### Обновления приложения

В **Настройки → Интеграции** укажите репозиторий **ljosefl/WPAdmin-Releases** — приложение сравнивает вашу версию с тегом последнего релиза на GitHub (формат тега: 1.0.1).
